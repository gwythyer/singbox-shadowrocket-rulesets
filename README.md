# singbox-shadowrocket-rulesets
Формирование rule-set для singbox (nekobox) и shadowrocket

Генерируются .srs и .list для конфигураций nekoray и shadowrocket

### Категории (direct и proxy):
- Anime
- Manga
- Ranobe
- Images
- Videos
- Games
- Software
- Socials
- Misc
- CDN
- Chatbots
- Adult Chatbots
- Hentai Images
- Hentai Manga
- Hentai Videos
- Hentai Games
- JAV
- Porn Images
- Porn Videos
- Porn Games

### Обновление
Релиз собирается автоматически при изменении файлов:
- rulesets/source/direct/*.txt
- rulesets/source/proxy/*.txt

### Использование
- nekoray → `rule_set`
- Shadowrocket → `RULE-SET`
